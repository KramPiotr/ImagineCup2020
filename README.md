## Installation

Install all the necessary packages using npm:

```bash
$ npm install
```

To Configure Redis and PostGres SQL Database, use `.env-example` to copy & paste the variables into your own `.env` file.

## Running the API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
